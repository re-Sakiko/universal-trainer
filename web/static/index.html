支持的格式

ShareGPT / OpenAI:
[{"messages": [{"role": "user", "content": "..."},
               {"role": "assistant", "content": "..."}]}]

Alpaca:
[{"instruction": "...", "input": "...", "output": "..."}]

JSONL:
{"question": "...", "answer": "..."}
{"prompt": "...", "response": "..."}

CSV:
question,answer
"问题1","答案1"

快速开始

  1. 把基座模型放到 models/ 目录
  2. 把数据集放到 datasets/ 目录
  3. 左侧填入路径 → 准备训练 → 开始训练
  4. 训练完成后切换到推理测试 Tab 测试模型
